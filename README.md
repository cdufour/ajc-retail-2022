# Formation Docker

## Notes
Le dossier __htdocs__ sert de volume à associer à un conteneur apache par la commande suivante:  

```
docker run -d --name apache -p 8000:80 -v "$PWD/htdocs/":/usr/local/apache2/htdocs httpd:2.4-alpine
```
